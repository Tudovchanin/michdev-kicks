
# michdev-kicks  

**Nuxt 4 Fullstack Sneakers Store with Admin Panel**  

This is a modern fullstack e-commerce store focused on sneakers. The project includes a public store for customers and an admin panel for managing products, orders, and users.  


## Tech Stack  

- **Frontend & Backend:** [Nuxt 4](https://nuxt.com/)  
- **State Management:** [Pinia](https://pinia.vuejs.org/)  
- **Database:** SQLite (dev) / MySQL (prod) with [Prisma](https://www.prisma.io/) ORM  
- **Authentication:** Email-based passwordless login  
- **Styling:** SCSS / CSS
